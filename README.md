# Uniform Residential Appraisal Report (Form 1004)

A web-based implementation of the Fannie Mae Form 1004 / Freddie Mac Form 70 Uniform Residential Appraisal Report, built using [A2UI](https://github.com/google/A2UI) and styled with the Atlassian Design System.

## Features

- Complete URAR Form 1004 with 8 sections:
  - Subject Property Information
  - Contract Details
  - Neighborhood Analysis
  - Site Information
  - Improvements Description
  - Sales Comparison Approach
  - Reconciliation
  - Appraiser Certification
- Atlassian Design System theming
- Responsive layout with tabs navigation
- Form data persistence (save draft)

## Technology

- **A2UI** - Agent-to-User Interface specification by Google
- **Lit** - Fast, lightweight web components
- **Vite** - Next generation frontend tooling
- **TypeScript** - Type-safe JavaScript

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
